# Lever API Job Board Integration

This single-file HTML application demonstrates how to integrate Lever's public Postings API into a custom career page. It fetches live job listings, provides real-time filtering by department and location, and displays job details dynamically.

## Language

`html`

## How to Run

Double-click index.html to open it in any modern web browser, or serve it using a local HTTP server (e.g., 'python -m http.server 8000').
